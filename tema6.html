Тема 6. Перебір масиву, об'єкта і елементів

Теорія — $.each та .each()

У бібліотеці jQuery є дві різні сутності з назвою each:

Сутність Синтаксис Призначення Параметри функції
$.each()
(глобальна функція)		 $.each(array|object, fn) Універсальна функція jQuery — перебирає елементи масиву або властивості об'єкта index — індекс або ключ
value — значення
.each()
(метод jQuery)		 $(selector).each(fn) Метод jQuery-об'єкта — застосовується до набору DOM-елементів для організації циклу по них index — порядковий номер ітерації
element — DOM-елемент (= this)

Синтаксис $.each():

$.each(array_або_object, function(index, value) {
    // index  — індекс (для масиву) або ключ (для об'єкта)
    // value  — значення поточного елемента
    // this   — посилання на поточний елемент
});

Синтаксис .each():

$(selector).each(function(index, element) {
    // index   — порядковий номер поточної ітерації (з 0)
    // element — DOM-елемент (аналог this)
    // this    — посилання на поточний DOM-елемент
});

Переривання та пропуск ітерації:

АналогВ $.each / .each()Опис
break return false; Переривання циклу $.each — повернути false
continue return; Перехід до наступної ітерації — повернути будь-яке значення, відмінне від false

Завдання 1 — $.each та .each() для елементів з class="name"

Організувати цикл each за елементами DOM, що мають клас name, двома способами, використовуючи дві різні сутності each: $.each($('.name')...) та $('.name').each(...).

Спосіб 1 — $.each($('.name'), function(index, element) {}):

$.each($('.name'), function(index, element) {
    console.log('index: ' + index + ', text: ' + $(element).text());
});

Спосіб 2 — $('.name').each(function(index, element) {}):

$('.name').each(function(index, element) {
    console.log('index: ' + index + ', text: ' + $(element).text());
});

Перебрати всі дочірні елементи, розташовані в ul з id "myList".

Завдання 3 — Визначити останній індекс та елемент в <div class="name">

Визначити останній індекс та останній елемент у колекції div.name.

var lastIndex = -1;
var lastText  = '';

$('div.name').each(function(index) {
    lastIndex = index;
    lastText  = $(this).text();
});
