Функція noConflict()

Деякі бібліотеки JavaScript також використовують символ $. Щоб уникнути конфліктів, існує функція noConflict() — вона звільняє символ $ і дозволяє використовувати інше ім'я для jQuery.

var jq = $.noConflict();
jq(document).ready(function () {
    jq('p').css('color', 'blue');
});

Ланцюжки команд (Method Chaining)

Команди в ланцюжку виконуються по черзі зліва направо — це дозволяє скорочувати розмір коду:

$('#chainDemo')
    .css('color', 'white')
    .css('background', '#1976d2')
    .text('Стилі змінено ланцюжком!')
    .animate({ opacity: 0.7 }, 500);

Цей абзац буде змінено ланцюжком методів
