Метод .filter() — 4 варіанти використання

Метод .filter() проводить фільтрацію набору елементів за допомогою заданого параметру.

// Варіант 1: filter(selector)
$('li').filter(':odd').css('background', '#bbdefb');

// Варіант 2: filter(function)
$('li').filter(function(index) {
    return index % 3 === 0;   // true → елемент потрапляє у результат
}).css('font-weight', 'bold');

// Варіант 3: filter(DOM-елемент)
var el = document.querySelector('#list li:first-child');
$('li').filter(el).css('border', '2px solid red');

// Варіант 4: filter(jQuery-набір)
var $set = $('#list li').eq(2);
$('li').filter($set).css('color', 'purple');

Практичний приклад усіх 4 варіантів:

  • Елемент 1
  • Елемент 2
  • Елемент 3
  • Елемент 4
  • Елемент 5
  • Елемент 6

Завдання — використання методу .filter()

Початковий код сторінки. Використовуючи метод .filter(), необхідно:

  1. Виділити текст між тегами <span></span> — червоний колір на жовтому фоні
  2. Текст між тегами <span id="first"></span> — 18px та підкреслений
  3. Текст між тегами <span id="second"></span> — 16px та напівжирний
  4. Текст між тегами <span id="city"></span> — 20px та нахилений
  5. В першому списку id="list" — виділити всі парні записи; перший і останній пункт — жирним шрифтом
  6. В другому списку id="two" — виділити всі непарні записи; перший і останній пункт — жирним шрифтом

Загальний текст між звичайним span на сторінці. Назва закладу: Черкаський державний бізнес-коледж. Тип закладу: вищий навчальний заклад. Місто: Черкаси.

Завдання 1 Всі <span> → .filter('span') → червоний на жовтому
Завдання 2 <span id="first"> → .filter('#first') → 18px, підкреслений
Завдання 3 <span id="second"> → .filter('#second') → 16px, напівжирний
Завдання 4 <span id="city"> → .filter('#city') → 20px, нахилений

Список 1 id="list" (парні — підсвічуються зеленим; перший і останній — жирні):

  • Перший елемент
  • Другий елемент
  • Третій елемент
  • Четвертий елемент
  • П'ятий елемент
  • Шостий елемент

Список 2 id="two" (непарні — підсвічуються блакитним; перший і останній — жирні):

  • Елемент A
  • Елемент B
  • Елемент C
  • Елемент D
  • Елемент E
  • Елемент F
