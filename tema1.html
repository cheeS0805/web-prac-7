Завдання 1.1 — Підключення jQuery через Google CDN

На сторінці заданий jQuery код, але він не працює, тому що не підключений файл бібліотеки. Підключіть jquery.js віддалено (скориставшись послугою Google).

Рішення — додати в <head>:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>

Взаємодія: alert(), prompt(), confirm()

Розглянемо три стандартні функції браузера для взаємодії з користувачем:

alert() — показує повідомлення

Синтаксис: alert("повідомлення"); — зупиняє виконання скриптів до натискання OK.

prompt() — запитує введення тексту

Синтаксис: prompt("текст", "значення за замовчуванням"); — повертає рядок або null.

confirm() — запитує підтвердження (OK / Скасування)

Синтаксис: confirm("питання"); — повертає true або false.

Підключення плагіну jquery-confirm

Офіційний сайт: https://craftpip.github.io/jquery-confirm

Плагін замінює стандартні модальні вікна на стилізовані. Підключення (Рис. 1.10):

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/jquery-confirm/3.3.4/jquery-confirm.min.css">

<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery-confirm/3.3.4/jquery-confirm.min.js"></script>

$.alert() — додає одну кнопку (OK), якщо кнопки не вказані.
$.confirm() — якщо кнопки не вказані, будуть додані дві (Okay & Cancel).

$.fn.confirm() — пряма прив'язка до елемента (Рис. 1.11)

Можна використовувати безпосередньо для прив'язки до елемента. При кліку на елемент відкривається вікно підтвердження. Всередині обробника доступний this.$target — елемент, на який натиснули.

Якщо жодна кнопка не визначена — будуть додані кнопки за замовчуванням (Okay та Cancel). Для посилань дія за замовчуванням — перехід на href.
