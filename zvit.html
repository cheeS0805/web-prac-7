Тема 1. Підключення jQuery та плагін jquery-confirm

Завдання 1. Підключити jQuery до сторінки через Google CDN та перевірити коректність підключення.
Завдання 2. Підключити плагін jquery-confirm і реалізувати діалогові вікна з кнопками.
Завдання 3. Створити власний jQuery-плагін $.fn.confirm — підтвердження дій з кнопками «Так» / «Ні».

Підключення jQuery через Google CDN:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>

Підключення плагіну jquery-confirm:

<link   rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/jquery-confirm/3.3.4/jquery-confirm.min.css">
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery-confirm/3.3.4/jquery-confirm.min.js"></script>

Перевірка підключення jQuery:

if (typeof jQuery !== 'undefined') {
    $('#result').text('jQuery v' + $.fn.jquery + ' підключено!');
}

Власний плагін $.fn.confirm з this.$target:

$.fn.confirm = function(options) {
    var $btn = this;
    var settings = $.extend({ title: 'Підтвердити?', message: '' }, options);
    $btn.on('click', function() {
        $.confirm({
            title: settings.title,
            content: settings.message,
            buttons: {
                yes: { text: 'Так', action: function() { settings.onYes && settings.onYes($btn); } },
                no:  { text: 'Ні',  action: function() {} }
            }
        });
    });
    return this;
};

noConflict() та ланцюжок методів:

Для демонстрації noConflict() використано var jq = jQuery (без реального виклику $.noConflict(), щоб не порушити решту коду). Ланцюжок методів дозволяє застосовувати декілька дій до одного елемента в один рядок.

Практичне завдання — код:

// 1. Всі span → червоний текст, жовтий фон
$('#task-area span').css({ 'color': 'red', 'background-color': 'yellow' });

$('#task-area span').filter('#first').css({ 'font-size': '18px', 'text-decoration': 'underline' });

// 3. span#second → розмір 16px, напівжирний
$('#task-area span').filter('#second').css({ 'font-size': '16px', 'font-weight': 'bold' });

// 4. span#city → розмір 20px, нахилений
$('#task-area span').filter('#city').css({ 'font-size': '20px', 'font-style': 'italic' });

$('#list li').filter(':first').css('font-weight', 'bold');
$('#list li').filter(':last').css('font-weight', 'bold');

// 6. Непарні li у #two → блакитний фон; перший і останній → жирний
$('#two li').filter(':odd').css('background-color', '#bbdefb');
$('#two li').filter(':first').css('font-weight', 'bold');
$('#two li').filter(':last').css('font-weight', 'bold');

Завдання 3. Два об'єкти з параметрами студента — порівняти поверхневе та глибоке $.extend(), вивести таблицю зі значеннями до та після злиття.

Порівняння методів теми 5:

МетодСинтаксисПризначення
$.map()$.map(array, fn)Перетворює кожен елемент масиву; повертає новий масив
$.merge()$.merge(arr1, arr2)Дописує елементи arr2 в кінець arr1 (змінює arr1)
$.extend()$.extend([deep], target, obj…)Копіює властивості об'єктів у target; deep=true — рекурсивне злиття
jQuery.unique()jQuery.unique(array)Видаляє дублікати DOM-елементів із масиву

Завдання 1 — $.merge та сортування:

// Об'єднання двох груп
var all = $.merge($.merge([], group1), group2);

// Сортування за прізвищем
all.sort(function(a, b) { return a.surname.localeCompare(b.surname, 'uk'); });

// Сортування за віком
all.sort(function(a, b) { return a.age - b.age; });

Завдання 2 — $.map з порядковими номерами:

// Додати порядковий номер кожному студенту
var numbered = $.map(all, function(s, i) {
    return $.extend({}, s, { num: i + 1 });
});

// Студенти з парним порядковим номером
var even = $.grep(numbered, function(s) { return s.num % 2 === 0; });

// Верхній регістр прізвищ
var upper = $.map(all, function(s) {
    return $.extend({}, s, { surname: s.surname.toUpperCase() });
});

Завдання 3 — поверхневе vs глибоке $.extend:

// Поверхневе злиття — вкладені об'єкти замінюються, а не зливаються
var shallow = $.extend({}, obj1, obj2);

// Глибоке злиття — рекурсивне об'єднання вкладених об'єктів
var deep = $.extend(true, {}, obj1, obj2);

Висновки (Теми 1–10)

У ході виконання практичних завдань тем 1–10 опановано основи роботи з бібліотекою jQuery. Підключення через Google CDN та jquery-confirm плагін (Тема 1) показало зручність використання готових компонентів. Власний плагін $.fn.confirm продемонстрував механізм розширення jQuery.

Теми 2–3 охопили роботу з селекторами (по id, класу, атрибутах, дочірні) та фільтрами (9 базових, 10 контентних, 15 фільтрів форм). Метод .filter() дозволяє звужувати вибірку за будь-яким критерієм: рядком-селектором, функцією, DOM-елементом або jQuery-об'єктом.

Теми 4–5 охопили роботу з масивами: $.makeArray() конвертує NodeList у звичайний масив, $.inArray() шукає елемент, $.grep() фільтрує за умовою, $.map() трансформує, $.merge() об'єднує масиви, а $.extend() зливає об'єкти — поверхнево або рекурсивно.

Тема 6 закріпила роботу з $.each() та .each() — перебір масивів, об'єктів та DOM-елементів. Тема 7 охопила систему обробки подій: методи .on(), .off(), .one(), скорочені обробники та делегування подій для динамічно доданих елементів. Тема 8 — анімація: методи .hide(), .show(), .fadeTo() та гнучкий .animate() з чергою анімацій.

Тема 9 охопила зміну вмісту елементів: .html() для вставки HTML-розмітки, .text() для безпечної роботи з текстом (з автоматичним екрануванням тегів), та .val() для роботи з елементами форм — текстовими полями, чекбоксами, радіо-кнопками та випадаючими списками з множинним вибором.

Тема 10 завершила вивчення методичного курсу: .attr() для читання та зміни HTML-атрибутів (в т.ч. об'єктний синтаксис і функціональний синтаксис для масових операцій), .removeAttr() для видалення атрибутів, та .prop() для роботи з булевими DOM-властивостями (disabled, checked, selected).
