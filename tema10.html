Тема 10. Методи jQuery для виконання операцій над атрибутами

Теорія — Методи .attr() та .removeAttr()

Синтаксис Режим Опис
$(sel).attr(name) Getter Повертає значення атрибута name першого елемента з набору
$(sel).attr(name, value) Setter Встановлює значення атрибута name для всіх елементів набору
$(sel).attr(name, function(i, val){}) Setter (fn) Встановлює атрибут за допомогою функції. Параметри: i — індекс, val — поточне значення атрибута
$(sel).attr({ name: val, ... }) Setter (obj) Встановлює кілька атрибутів одразу через об'єкт
$(sel).removeAttr(name) Remove Видаляє атрибут name з усіх елементів набору

Приклади .attr():

// Getter — отримати значення
var src = $('img:first').attr('src');
var alt = $('img:first').attr('alt');

// Setter — встановити одне значення
$('img:first').attr('title', 'Підказка для зображення');

// Setter — встановити кілька атрибутів одразу
$('img:first').attr({ alt: 'Ноутбук', title: 'Фото ноутбука' });

// Setter з функцією — використати поточне значення
$('img').attr('title', function(index, currentAlt) {
    return $(this).attr('alt') + ' (зображення ' + (index + 1) + ')';
});

// Видалити атрибут
$('img:first').removeAttr('title');

Теорія — Метод .prop() та різниця між .attr() та .prop()

Метод .prop() працює з DOM-властивостями (property), а .attr() — з HTML-атрибутами (attribute). Для більшості атрибутів різниця несуттєва, але для булевих вона принципова.

СинтаксисРежимОпис
$(sel).prop(name) Getter Повертає значення DOM-властивості (наприклад, true/false для checked)
$(sel).prop(name, true|false) Setter Якщо true — атрибут встановлюється, якщо false — видаляється
Критерій.attr().prop()
Що повертає для checked "checked" або undefined true або false
Після зняття галочки Все одно "checked" (початковий стан) Повертає false (актуальний стан)
Призначення HTML-атрибути: href, src, alt, class Булеві властивості: checked, disabled, selected

Приклади .prop():

// Перевірити стан чекбокса
var isChecked = $('input[type="checkbox"]').prop('checked'); // true / false

// Вимкнути кнопку
$('#myButton').prop('disabled', true);   // додає disabled

// Увімкнути кнопку
$('#myButton').prop('disabled', false);  // видаляє disabled

// Відмітити чекбокс
$('#myCheckbox').prop('checked', true);

Демонстраційні зображення для завдань 1–4

На сторінці є 5 зображень. Кожне має атрибут alt, але жодне не має title (поки не виконано завдання).

img#1 — alt: "Ноутбук"
img#2 — alt: "Смартфон"
img#3 — alt: "Планшет"
img#4 — alt: "Монітор"
img#5 — alt: "Клавіатура"

Поточний стан атрибутів:

#idalttitle

Завдання 1 — Отримати alt першого зображення і встановити його як title

Отримати значення атрибута alt першого зображення на сторінці і, використовуючи його, встановити значення атрибута title.

// Отримати alt першого зображення (getter)
var altValue = $('img.demo-img:first').attr('alt');

// Встановити title = alt (setter)
$('img.demo-img:first').attr('title', altValue);

console.log('alt   = ' + altValue);
console.log('title = ' + $('img.demo-img:first').attr('title'));

Завдання 2 — Додати атрибути alt і title першому зображенню

Додати атрибути alt і title першому зображенню одночасно, використовуючи об'єктний синтаксис .attr({}).

// Встановити кілька атрибутів одразу через об'єкт
$('img.demo-img:first').attr({
    'alt'  : 'Зображення ноутбука',
    'title': 'Сучасний ноутбук'
});

Завдання 3 — Встановити значення атрибутів alt і title для першого зображення

Встановити значення атрибутів alt і title для першого зображення за допомогою ланцюжка викликів .attr().

// Ланцюговий виклик .attr() для встановлення alt і title окремо
$('img.demo-img:first')
    .attr('alt',   'Портативний комп\'ютер')
    .attr('title', 'Зображення 1 — Портативний комп\'ютер');

Завдання 4 — Встановити title = alt для кожного зображення

Встановити значення атрибута title, використовуючи значення атрибута alt, для кожного зображення на сторінці. Застосовується функціональний синтаксис .attr(name, fn).

// Для кожного img встановити title на основі його поточного alt
// Функція викликається для кожного елемента з індексом i та поточним значенням val
$('img.demo-img').attr('title', function(i, val) {
    return $(this).attr('alt');
});
