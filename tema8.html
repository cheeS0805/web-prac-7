Тема 8. Анімація в jQuery

Теорія — Методи показу та приховування

Для показу та приховування елементів jQuery надає методи .hide(), .show() та .toggle().

Метод Синтаксис Опис
.hide() $(selector).hide([duration] [, easing] [, callback]) Приховує вибрані елементи (встановлює display:none). Без параметрів — миттєво.
.show() $(selector).show([duration] [, easing] [, callback]) Показує приховані елементи. Без параметрів — миттєво.
.toggle() $(selector).toggle([duration] [, easing] [, callback]) Перемикає між показом та прихованням елементів.

Параметри .hide() / .show() / .toggle():

ПараметрТипЗначення за замовчуваннямОпис
duration Number або String 400 Тривалість анімації (мс). Рядки: "fast" = 200 мс, "slow" = 600 мс.
easing String "swing" Функція плавності. "swing" — повільніше на початку та в кінці, швидше в середині. "linear" — рівномірна швидкість.
callback Function — Функція, що викликається після завершення анімації.

Приклад:

$('#myDiv').hide(1000);              // приховати за 1 секунду
$('#myDiv').show('slow');             // показати повільно (600 мс)
$('#myDiv').toggle('fast', function() {
    console.log('Анімація завершена');
});

Теорія — Методи зміни прозорості (fade)

Метод Синтаксис Опис
.fadeIn() $(sel).fadeIn([duration] [, easing] [, callback]) Плавно збільшує прозорість до видимого стану (opacity: 0 → 1). Встановлює display:block на початку.
.fadeOut() $(sel).fadeOut([duration] [, easing] [, callback]) Плавно зменшує прозорість до прихованого (opacity: 1 → 0). Після завершення встановлює display:none.
.fadeTo() $(sel).fadeTo(duration, opacity [, easing] [, callback]) Плавно змінює прозорість до необхідного рівня. На відміну від fadeOut не приховує елемент після завершення.
.fadeToggle() $(sel).fadeToggle([duration] [, easing] [, callback]) Перемикає між fadeIn та fadeOut.

Параметри .fadeTo():

ПараметрТипОбов'язковийОпис
duration Number або String Так Тривалість анімації в мілісекундах або "fast" / "slow".
opacity Number Так Цільове значення прозорості від 0 (повністю прозорий) до 1 (повністю непрозорий).
easing String Ні Функція плавності: "swing" або "linear".
callback Function Ні Функція зворотного виклику після завершення анімації.

Приклад:

$('#myDiv').fadeOut(500);            // сховати за 0.5 с
$('#myDiv').fadeIn('slow');           // показати повільно
$('#myDiv').fadeTo(2000, 0.3);        // знизити прозорість до 30% за 2 с

Теорія — Метод .animate()

Метод .animate() є найбільш гнучким методом анімації jQuery. Він дозволяє анімувати будь-яке числове CSS-властивість.

Синтаксис:

$(selector).animate(properties [, duration] [, easing] [, callback]);
// або через об'єкт опцій:
$(selector).animate(properties, options);

Параметри .animate():

ПараметрТипЗначення за замовч.Опис
properties PlainObject — Об'єкт з CSS-властивостями та їх цільовими значеннями. Властивості — camelCase. Значення — числові або спеціальні рядки: "hide", "show", "toggle". Допускаються відносні значення з += або -=.
duration Number або String 400 Тривалість анімації (мс). "fast" = 200 мс, "slow" = 600 мс.
easing String "swing" Крива швидкості анімації (кубічна крива Безьє). "linear" — рівномірна швидкість від початку до кінця. "swing" — повільний старт і закінчення, швидкість збільшується в середині.
complete Function — Функція, яка буде виконана після завершення анімації. Викликається один раз для кожного відповідного елемента. Всередині функції this посилається на DOM-елемент.
queue Boolean або String true Якщо false — анімація виконується негайно, поза чергою. Рядок — назва черги (з jQuery 1.7). При користувацькому імені черги необхідно викликати .dequeue("ім'я").
specialEasing PlainObject — Об'єкт, що містить одну або кілька CSS-властивостей і відповідні їм функції плавності. Додано у версії 1.4.
step Function(now, tween) — Функція, що викликається для кожної анімованої властивості кожного анімованого елемента. Дає можливість змінювати об'єкт Tween перед встановленням значення.
progress Function — Функція, що викликається після кожного кроку анімації, лише один раз для кожного анімованого елемента (незалежно від кількості властивостей). Додано у версії 1.8.
start Function(animation) — Функція, що викликається, коли анімація елемента починається. Додано у версії 1.8.
done Function(animation, jumpedToEnd) — Функція, що викликається, коли анімація елемента завершується успішно. Додано у версії 1.8.
fail Function(animation, jumpedToEnd) — Функція, що викликається, коли анімацію елемента не вдається завершити. Додано у версії 1.8.
always Function(animation, jumpedToEnd) — Функція, що викликається, коли анімація елемента завершується або зупиняється незавершеною. Додано у версії 1.8.

Увага! Метод .animate() не підтримує анімацію кольорів (color, backgroundColor) без плагіну jQuery UI.

Приклади:

// Проста анімація
$('#box').animate({ left: '250px', opacity: 0.5 }, 1000);

// Черга анімацій (виконуються послідовно)
$('#box').animate({ left: '300px' }, 1000)
         .animate({ top: '100px' }, 500)
         .animate({ opacity: 0 }, 800);

// Спеціальні значення
$('#box').animate({ height: 'toggle' });  // показати або приховати

Завдання 1 — .hide() та .show()

При натисканні на кнопку з id = but1 абзац з id = par1 повинен зникати, а при натисканні на кнопку c id = but2 — з'являтися.

Навчальний заклад було відкрито в 1966 році відповідно до Постанови Ради Міністрів СРСР від 07 листопада 1963 р. № 1210 на виробничій базі Черкаського хімічного комбінату (зараз – ПАТ Азот). Перша назва – технічне училище № 2 м. Черкаси.

$('#but1').click(function() {
    $('#par1').hide();
});

$('#but2').click(function() {
    $('#par1').show();
});

Завдання 2 — .fadeTo() при наведенні курсора

При наведенні курсора миші на абзац з id = par2 прозорість абзацу повинна плавно змінитися протягом 3 секунд, а при відведенні — відновитися.

В подальшому заклад реорганізовувався: у 1984 р. – у середнє професійно-технічне училище № 10 м. Черкаси, у 1991 р. – вище професійне училище № 10 м. Черкаси. 1985 року добудовано навчальний корпус № 2, у 2011р. завершено реконструкцію навчального корпусу №1.

$('#par2').mouseenter(function() {
    $(this).fadeTo(3000, 0.3);
});

$('#par2').mouseleave(function() {
    $(this).fadeTo(1000, 1);
});

Завдання 3 — .animate() зміна висоти

При натисканні на кнопку з id = but3 висота елемента wrap1 повинна бути зменшена до 0 протягом 5 секунд. При натисканні на кнопку but4 — повернутися до початкового значення за 7 секунд.

Перші 25 років своєї діяльності навчальний заклад займався підготовкою фахівців, в основному, для ПАТ Азот. Найбільш відомими випускниками тих років були Корчака Микола Іванович (головний інженер ПАТ Азот), Шепель Володимир Петрович (голова Ради Черкаської обласної профспілки працівників хімічної і нафтохімічної галузей промисловості), Шолудько Петро Іванович (начальник цеху контрольно-вимірювальних приладів і автоматики ПАТ Азот), Калько Тетяна Георгіївна (майстер виробничого навчання технічного училища № 2 м. Черкаси).

var wrap1Height = $('#wrap1').height();  // зберігаємо початкову висоту

$('#but3').click(function() {
    $('#wrap1').animate({ height: 0 }, 5000);
});

$('#but4').click(function() {
    $('#wrap1').animate({ height: wrap1Height }, 7000);
});
