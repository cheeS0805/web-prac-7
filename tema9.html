Тема 9. Зміна вмісту елементів за допомогою jQuery. Додавання вмісту на сторінку

Теорія — Методи .html(), .text(), .val()

jQuery надає три основні методи для роботи з вмістом елементів:

Метод Синтаксис (getter) Синтаксис (setter) Призначення
.html() $(sel).html() $(sel).html(str) Отримати або встановити HTML-вміст елемента (включно з тегами)
.text() $(sel).text() $(sel).text(str) Отримати або встановити текстовий вміст елемента (без тегів, HTML-теги екрануються)
.val() $(sel).val() $(sel).val(str|array) Отримати або встановити значення атрибута value елементів форми

Порівняння .html() та .text():

Критерій.html().text()
HTML-теги у вмісті Вставляються як HTML (інтерпретуються браузером) Вставляються як текст (відображаються як <b>, & тощо)
Отримання значення Повертає HTML разом з тегами Повертає лише текстовий вміст (без тегів)
Безпека Небезпечно для введення користувача (XSS) Безпечно — вміст екранується
Застосування Зміна структури DOM, вставка розмітки Читання/запис тексту без вставки HTML

Метод .val() — типи елементів:

ЕлементGetterSetter
<input type="text"> Повертає поточне значення поля Встановлює значення поля
<input type="checkbox"> Повертає значення атрибута value Якщо передати масив — відмічає чекбокси, чиї value входять до масиву
<input type="radio"> Повертає значення відміченого radio Встановлює відмічений radio за значенням
<select> Повертає вибране значення (або масив для multiple) Рядок — вибирає один варіант; масив — вибирає кілька (multiple)
<textarea> Повертає вміст текстового поля Встановлює вміст текстового поля

Приклади:

// .html() — getter та setter
var content = $('#myDiv').html();           // отримати HTML-вміст
$('#myDiv').html('<b>Новий вміст</b>');    // встановити HTML

// .text() — getter та setter
var text = $('p').text();                  // отримати текст усіх p
$('p').text('Новий текст');               // встановити текст усіх p

// .val() — getter та setter
var inputVal = $('input').val();           // отримати значення
$('input').val('Нове значення');          // встановити значення
$('select').val(['Option1', 'Option3']);   // вибрати кілька (multiple)
$('[name="cb"]').val(['val1', 'val2']);    // відмітити чекбокси

Завдання 3 — Метод .val()

Використовуючи метод .val(), при натисканні на кнопку:

  • змінюємо значення текстових полів <input> у div.firstTest
  • відмічаємо необхідні елементи <input> у div.secondTest
  • вибираємо необхідні елементи списку <select>


$('#btn3').click(function() {
    // Змінюємо значення текстових полів у .firstTest
    var inputs = $('#task3-area .firstTest input[type="text"]');
    inputs.eq(0).val('changedFirst');
    inputs.eq(1).val('changedSecond');

    // Відмічаємо чекбокси у .secondTest (масив значень)
    $('[name="checkboxname"]').val(['check1', 'check2']);

    // Вибираємо radio у .secondTest
    $('[name="radio"]').val(['radio2']);

    // Вибираємо елементи списку <select>
    $('#task3-area select').val(['Option2', 'Option4']);

    // Виводимо результат
    var res = 'input[0].val = "' + inputs.eq(0).val() + '"\n';
    res    += 'input[1].val = "' + inputs.eq(1).val() + '"\n';
    res    += 'radio.val = "'   + $('[name="radio"]:checked').val() + '"\n';
    res    += 'select.val = '   + JSON.stringify($('#task3-area select').val());
    $('#task3result').text(res).show();
});
