Тема 5. Масиви jQuery — сортування та об'єднання

Теорія: методи сортування та об'єднання

$.map(array/object, function) — перетворення елементів

Застосовує задану функцію до кожного елемента масиву або властивості об'єкта та повертає новий масив. Функція обробляє кожен елемент і повертає нове значення.

var arr = [1, 2, 3, 4, 5];
var doubled = $.map(arr, function(val, i) {
    return val * 2;
});
// → [2, 4, 6, 8, 10]

jQuery.unique(array) — видалення дублікатів DOM-елементів

Функція jQuery.unique() сортує масив DOM-елементів та видаляє дублікати. Вона працює виключно з DOM-елементами — не з рядками чи числами.

var divs = $('div').get();          // масив DOM-елементів
var unique = jQuery.unique(divs);    // видалити дублікати
console.log(unique.length);          // кількість унікальних div

$.merge(first, second) — об'єднання двох масивів

Об'єднує вміст другого масиву у перший. Змінює перший масив та повертає його. Для збереження оригіналу — передавати порожній масив першим аргументом.

var arr1 = [1, 2, 3];
var arr2 = [4, 5, 6];
$.merge(arr1, arr2);   // arr1 = [1, 2, 3, 4, 5, 6]
// Безпечне злиття (не змінює оригінали):
var merged = $.merge($.merge([], arr1), arr2);

$.extend([deep], target, object1, [object2, ...]) — розширення об'єктів

Зливає властивості одного або більше об'єктів у цільовий об'єкт.

  • Перший параметр — цільовий об'єкт (він буде змінений і повернутий)
  • Якщо перший аргумент — true, виконується глибоке злиття
  • При поверхневому злитті вкладені об'єкти перезаписуються повністю
  • При глибокому злитті вкладені об'єкти зливаються рекурсивно
// Поверхневе злиття:
var result = $.extend({}, obj1, obj2);
// Глибоке злиття:
var deep = $.extend(true, {}, obj1, obj2);
