Тема 4. Масиви в jQuery — створення і маніпуляції

Теорія: функції роботи з масивами jQuery

$.makeArray(obj) — конвертація масиво-подібних об'єктів у масив

Перетворює масиво-подібний об'єкт (NodeList, arguments тощо) у справжній JavaScript-масив, до якого можна застосовувати методи sort(), pop(), reverse().

var liNodes = document.querySelectorAll('li');  // NodeList — не масив
var arr     = $.makeArray(liNodes);              // тепер справжній Array

$.inArray(value, array) — пошук значення в масиві

Повертає індекс першого збігу або -1 якщо не знайдено. Аналог Array.indexOf().

var names = ['Іваненко', 'Петренко', 'Сидоренко'];
$.inArray('Петренко', names);  // → 1
$.inArray('Шевченко', names);  // → -1

$.grep(array, function, [invert]) — фільтрація масиву

Відбирає елементи масиву за умовою. Параметри функції-фільтра:

  • element — поточний елемент масиву
  • index — індекс поточного елемента
  • invert (необов'язковий) — якщо true, відбирає елементи, для яких функція повернула false
var nums = [1, 2, 3, 4, 5, 6];
$.grep(nums, function(val) { return val > 3; });         // → [4, 5, 6]
$.grep(nums, function(val, i) { return i % 2 === 0; }); // → [1, 3, 5] (парні індекси)
